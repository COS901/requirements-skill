## 📄 About requirements-skill

requirements-skill helps you manage product requirements using AI. It detects conflicts in requirements, extracts change logs, and offers example stories for software-as-a-service (SaaS) products. This makes it easier to keep your project organized and up to date.

The software works on Windows and does not require any programming knowledge.

## 🖥️ System Requirements

- Windows 10 or later (64-bit recommended)  
- At least 4 GB of RAM  
- 200 MB of free disk space  
- Internet connection to download and use AI features  

## ⚡ Features

- Detects conflicts in product requirements automatically.  
- Extracts change logs from updates to help track progress.  
- Provides example stories tailored for SaaS projects.  
- Works with plain text files for requirements.  
- Simple to use with no coding necessary.

## 🔍 How To Use Key Features

### Conflict Detection

The app scans your requirements for conflicts such as contradictory statements or overlapping features.  

Conflicts appear highlighted in red. Hover your mouse over the highlighted areas for more information.

### Change Log Extraction

When you update your requirement files, the app compares versions and extracts change logs.  

You will see summaries of what was added, removed, or edited. This helps keep track of project updates.

### Example Stories for SaaS

The software provides sample user stories that match common SaaS scenarios.  

You can select and insert these stories into your documents to clarify requirements.
